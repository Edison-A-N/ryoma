## ✨ 功能特点

- 🧠 基于 LangGraph 构建的智能代理系统
- 🔌 支持多个 LLM 提供商:
  - OpenAI
  - Google PaLM
  - AWS
- ⚡️ 高性能 FastAPI 后端

- 🐍 Python 3.11+
- 🔄 LangGraph
- ⚡ FastAPI
- ☁️ 云服务集成:
  - AWS (Boto3)
  - Google Cloud AI Platform
  - OpenAI API
